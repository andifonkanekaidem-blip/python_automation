File Organizer Script (Python)



A simple automation tool that organizes files into folders based on their extensions.



\## Features

\- Automatically sorts files into categories (Images, Documents, Videos, etc.)

\- Handles unknown file types by placing them in an "Others" folder

\- Works on Windows, macOS, and Linux

\- Clean and beginner-friendly code



\## How It Works

1\. The script scans a target folder

2\. Detects each file's extension

3\. Moves the file into the correct category folder

4\. Creates missing folders automatically



\## Technologies Used

\- Python `os`

\- Python `shutil`

\- Basic file handling



\## Example

Before:



Downloads/ image1.jpg doc.pdf video.mp4 report.xlsx



After:



Downloads/ Images/ Documents/ Videos/ Others/



\## How to Use

1\. Put the script in any folder

2\. Run:



python file\_organizer.py



3\. Copy the path to the folder you wish to sort.


4\. Your files will be sorted instantly.



\## Perfect For

\- People who want clean folders  

\- Automating repetitive file management  

\- Learning Python automation basics
